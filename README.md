- Memahami fundamental JavaScript secara kuat
- Meningkatkan kemampuan problem solving

### 1. Beginner
Berisi materi dan project dasar JavaScript.

Fokus pembelajaran:
- Variabel (let, const)
- Tipe data (string, number, boolean, array, object)
- Operator
- Percabangan (if-else, switch)
- Perulangan (for, while)
- Function dasar
- DOM dasar

Contoh project:
- Kalkulator sederhana
- To-do list sederhana
- Counter app
- Game tebak angka

---

### 2. Proficient
Level menengah â€” mulai berpikir seperti developer.

Fokus pembelajaran:
- Array methods (map, filter, reduce)
- Object manipulation
- ES6+ (arrow function, destructuring, spread operator)
- Async JavaScript (Promise, async/await)
- Fetch API
- Modular JavaScript
- Error handling

Contoh project:
- Weather App (API)
- Notes App dengan Local Storage
- Authentication sederhana
- CRUD sederhana

---

### 3. Expert
Level lanjutan â€” mendekati level production-ready.

Fokus pembelajaran:
- Clean Code & Best Practice
- Design Pattern dasar
- OOP & Functional Programming
- Performance optimization
- Security awareness (basic frontend security)
- Testing dasar
- Struktur project scalable

Contoh project:
- Mini e-commerce frontend
- Dashboard dengan API
- Real-time app (WebSocket)
- Fullstack integration

---

## Target Pembelajaran

Setelah menyelesaikan semua level, diharapkan mampu:

- Memahami JavaScript secara mendalam
- Membangun aplikasi tanpa tutorial step-by-step
- Membaca dokumentasi resmi
- Debugging secara mandiri
- Menulis kode yang bersih dan terstruktur
